k60烧写调试遇到的问题总结

Author: Jorwnpay Date: June 13, 2019  13:49:45 Category: 问题记录

烧写调试遇到的问题总结

  • 问题一
  • 报错：* Fatal Error[Lc002]: could not open file “D:\IAR\arm\config\config files\vcan_ROM_K60N512.icf”
  • 原因：* 路径配置错误，在当前路径下找不到vcan_ROM_K60N512.icf文件；
    这个文件在<–工程路径–>\Prj\IAR\config files\vcan_ROM_K60N512.icf路径下;
  • 解决方法：* 右键工程名，单击options…选项，在左手边选linker，右手边config下把下图所示的地方改成有vcan_ROM_K60N512.icf文件的路径。
  • 问题二
  • 报错：* Reading CPU status failed.
  • 原因：* 单片机被锁死
  • 解决方法：* 看山外这个论坛里的第四条，务必认真看，不要漏了操作步骤
    http://www.vcan123.com/forum.php?mod=viewthread&tid=84&ctid=1

    J-Link Commander的下载链接：
    http://www.vcan123.com/forum.php?mod=viewthread&tid=6233&ctid=2
    点里边的第一个链接

    下载完之后运行安装

    在开始界面就可以找到jlink commander。

2019.7.23更新

  • 问题三
  • 报错：* The flash loader program reported an error.
  • 原因：* 单片机被锁死
  • 解决方法：* 打开J-Link Commander输入unlock kinetis解锁即可。具体方法同问题二。
