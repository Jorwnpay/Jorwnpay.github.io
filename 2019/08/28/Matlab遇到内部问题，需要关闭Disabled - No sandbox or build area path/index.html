Matlab遇到内部问题，需要关闭Disabled - No sandbox or build area path

Author: Jorwnpay Date: August 28, 2019  22:20:17 Category: 问题记录

问题描述： 在使用matlab中的hist()函数画图的时候，报错说Disabled - No sandbox or build area path
详见下图：

我在MATLAB Answers网站上找到了相似的问题。

原因： 根据网友的经验，大概弄明白了原因：
有个老哥说之前有个相同的症状，造成的原因是英特尔特殊的图形驱动。

解决方法： 更新图形驱动
网友给出的一些解决经验：

驱动这东西有时候自己比较难搞，不妨用 驱动人生 这样的现成工具
虽然看上去很 low，但是这玩意儿从来没让我失望过。

更新完之后重启下电脑就好了
（如果没好再打开驱动人生看看有没有更新成功）
