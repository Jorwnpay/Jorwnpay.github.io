安装python的Anaconda发行版

Author: Jorwnpay Date: August 25, 2019  15:28:28 Category: 实践笔记

什么是Anaconda？

  • 英语层面的解释[1]
    Anaconda的意思是蟒蛇，python的意思也是蟒蛇，那么同样是蟒蛇，python和anaconda有什么区别呢？

    python [‘paɪθən] & anaconda [,ænə’kɑndə]
    Anacondas原产于南美洲，而Python则天然存在于亚洲和非洲的热带地区。
    相比之下，Anaconda更重，但python更长。
    作为蛇，Python比anaconda更灵活。
    Anaconda的着色图案被组织和安排得更有顺序，但python不是。
    Anaconda是一个很好的游泳运动员，经常在水边发现，而python喜欢栖息在树上和干燥的栖息地。
    Python作为宠物在人类中更受欢迎，但Anacondas通常不会被作为宠物饲养。
    Anaconda由雌性产出新生蛇是很典型的，而Python是下蛋并孵化出来。

  • 技术层面的解释
    顾名思义，Anaconda是python的一种，只是一个不同的发行版本。
    python拥有NumPy、SciPy、pandas、Matplotlib 和 scikit-learn 等众多功能齐全、接口统一的数据分析处理库，能为数据分析工作提供极大的便利。但是这些库的环境配置和版本问题及管理花费了分析人员大量的时间。为了避免上述问题，Anaconda发行版应运而生。

Anaconda发行版Python预装了150个以上的常用packages，囊括了数据分析常用的NumPy、SciPy、pandas、Matplotlib 、 scikit-learn 和 IPython 库，使得数据分析人员可以更加顺畅专注地使用python解决数据分析相关问题。[2]

windows系统下安装Anaconda

  • 进入Anaconda官方网站，下载对应版本的安装包
  • 运行安装文件

    Install for 选择 All Users

    后面保持默认就好
    等待安装完成
